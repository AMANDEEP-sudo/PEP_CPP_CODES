# Singly Circular Linked List Traversal

This program demonstrates insertion and traversal operations on a singly circular linked list in C++.

## File: circular_traversal.cpp

- Implements:
  - Node structure
  - Insert at head (linear and circular)
  - Display function

## Usage

Compile and run:

```
g++ circular_traversal.cpp -o circular_traversal.exe
./circular_traversal.exe
```
